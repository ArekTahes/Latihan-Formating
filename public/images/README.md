## Rekomendasi Ukuran Gambar

- Hero images: 1920x1080 px (16:9)
- OG image: 1200x630 px
- Gallery images: 800x800 px (1:1)
- Service images: 800x600 px (4:3)

## Tips

1. Gunakan format WebP untuk performa terbaik
2. Kompres gambar sebelum upload
3. Gunakan deskripsi yang jelas untuk SEO
